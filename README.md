# 40 LPA DevOps Portfolio

A hands-on DevOps / Platform / SRE portfolio focused on AWS, Terraform, Kubernetes, CI/CD, observability, and automation.

## Projects

### Week 1 - AWS 3-Tier Architecture with Terraform
Provisioned a modular AWS infrastructure using Terraform:
- VPC with public and private subnets
- Application Load Balancer
- Auto Scaling Group
- Bastion host
- RDS database
- Reusable Terraform modules

**Skills:** AWS, Terraform, Networking, High Availability, Infrastructure as Code

### Week 2 - CI/CD Deployment with GitHub Actions
Built a simple CI/CD pipeline that:
- Packages application code
- Connects to an EC2 instance
- Deploys app files to Apache web server
- Restarts web service automatically

**Skills:** GitHub Actions, Linux, SSH, SCP, CI/CD

### Week 3 - Kubernetes SRE Demo Platform
Production-style Kubernetes deployment for a Python app with:
- Dockerized service
- Deployment, Service, Ingress, ConfigMap
- Liveness and readiness probes
- Horizontal Pod Autoscaler
- Python health-check automation
- SRE-oriented design for reliability and scaling

**Skills:** Kubernetes, Docker, Python, SRE, Monitoring, Reliability Engineering

## What this portfolio demonstrates

- CI/CD automation
- Kubernetes workload deployment and scaling
- Python scripting for operational automation
- SRE practices like health checks, autoscaling, and service reliability
